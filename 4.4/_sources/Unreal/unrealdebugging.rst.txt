Debugging, Diagnosing and Profiling
====================================

trueSKY provides several systems for debugging and performance.

Console Commands
----------------

* **r.TrueSky.Memory**      -   Display trueSKY's memory usage onscreen.

* **r.TrueSky.CloudCrossSections**      -   Show cloud cross-sections as an overlay.

* **r.TrueSky.Compositing**     -   Show compositing elements as an overlay.

* **r.TrueSky.Profiling**   -   Display trueSKY's profiler onscreen.

* **r.TrueSky.DiagnosticMode**   -   Activate the diagnostic mode.
